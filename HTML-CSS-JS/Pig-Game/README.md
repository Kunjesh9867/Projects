# **Pig-Game**

## **Introduction**

<code>
This game can be played between 2 players. Two prompt windows will be appeared to write the player's name.
Now, the active player have to roll a dice.
After rolling a dice, the active player can either hold that number or continue to roll dice. If the next number on dice is 1 then all the current point will be reset to 0, and the opponent player will continue to play.
Whosever reaches 100 point, will win the game.

You can clone & run this game in your PC.</code>
